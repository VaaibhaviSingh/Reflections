Describe the differences between forks, clones, and branches.  When would you use one instead of another?

    Forks are to clone a repository on your GitHub server on GitHub itself. Clones are copy of repositories that can be made locally or can be on the GitHub server and be present locally. Branches are used to make new changes in a seperate path so that the main content does not gets effected.

    We can synchronize the changes made on the GitHub repository and our local repository in accordance with our needs.
